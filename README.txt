This is an interactive Retrieval-Augmented Generation (RAG) chatbot built with LangChain, Streamlit, and Groq LLMs. 
It allows you to upload PDFs and chat with them using memory-aware retrieval and a powerful LLM backend.

🚀 Features
🧠 Chat With PDFs – Ask context-aware questions about uploaded PDFs.

🧾 Chat Memory – Tracks and remembers previous conversations with session IDs.

⚡ Groq LLM (LLaMA 3.1) – Superfast responses via Groq’s low-latency language model.

🧱 ChromaDB – Local vector store for document chunk embeddings.

🔍 HuggingFace Embeddings – Semantic document understanding via all-MiniLM-L6-v2.

💬 Streamlit UI – Modern chat interface with styled layout and file uploads.

🛠️ Customizable Prompts – History-aware retriever and concise QA formatting.
